# NEOVIM-IDE
---
基于 neovim 打造编程开发，文档编写智能环境。
## windows 安装
### 依赖软件：
#### 程序依赖
- neovim for windows
- git for windows
- opencode
#### nodejs 依赖
- nodejs
  1. npm install -g @mermaid-js/mermaid-cli
  2. npm install -g neovim
#### scoop 依赖
- scoop
  1. lua51
  2. imagemagick
  3. ghostscript
  4. tectonic
  5. ripgrep
  6. fd
  7. main/mingw
#### python依赖
- python
  1. pip3 install pynvim
### 克隆仓库
打开 powershell，执行下述命令：

``` bash
git clone https://github.com/panderan/nvim-ide.git $env:LOCALAPPDATA\nvim
```
